# Vinnie's fancy String Calculator front-end
Using: 
- React
- TS
- Tailwind
- ShadcnUI
- Vite
- Cypress
- Vercel

Also: `vite-tsconfig-paths` to allow Vite to read tsconfig compiler option paths (so I don't need to define custom path resolution twice)
